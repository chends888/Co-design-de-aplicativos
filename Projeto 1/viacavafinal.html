Projeto final de Design de Software

O projeto final do curso de Design de Software foi o desenvolvimento de um aplicativo programado em Python utilizando a biblioteca Tkinter para a criação de interfaces gráficas. Seria necessário, além disso, utilizar alguma biblioteca que possibilitasse armazenar arquivos no computador local ou na rede, como Pickle ou Firebase. O aplicativo desenvolvido pelo meu grupo foi um organizador de churrasco que, a partir do número de pessoas e das comidas e bebidas selecionadas, calcula a quantidade de ítens que deve ser comprada. Embora com certos problemas, o aplicativo roda, mas não está pronto para ser lançado ao público.
